使用VsCode以及Github搭建个人图床
发表于2022-09-03|VsCode相关配置|图床
使用VsCode中的PicGo拓展以及Github搭建个人图床下载扩展在扩展商店搜索PicGO，安装即可 配置扩展 打开扩展设置 在设置中找到Picgo>Pic Bed: Uploader,在其中选中github 往上翻，找到Picgo>pic Bed:Current，设置以下属性: Branch: main(要看你仓库的分支) Custom Url: 加速链接,就是https://cdn.jsdelivr.net/gh/ 加上用户名/仓库名(现已失效) Path: 仓库下照片存储的路径,设置这个目录,随后会在仓库中自动生成该目录 Repo: 用户名/仓库名 踩坑指南 没错，这篇文章就是在VsCode中写的，用起来不错，但是问题也出现了，cdn加速链接已经失效了，这也就导致如果不挂梯，是看不到图片的····· 还可能会报服务端异常，这时检查你的Branch，看看是不是写错了，有的是master而不是main
在VsCode中集成Git-Bash
发表于2022-09-03|VsCode相关配置|VsCode
打开设置，输入 shell windows 打开settings.json在 “terminal.integrated.profiles.windows” 中添加以下字段123"Git-Bash": { "path": "D:\\java_instal\\Git\\bin\\bash.exe",} 新版的VSCODE这里是Git Bash,此时是识别不了的，下面的path是source，也是识别不了，可以直接删除原来的，添加新的就好了
