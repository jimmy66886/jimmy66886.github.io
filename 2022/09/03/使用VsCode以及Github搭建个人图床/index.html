使用VsCode中的PicGo拓展以及Github搭建个人图床

下载扩展

在扩展商店搜索PicGO，安装即可

配置扩展

  1. 打开扩展设置
  2. 在设置中找到Picgo>Pic Bed: Uploader,在其中选中github
    ![20220903174857]
  3. 往上翻，找到Picgo>pic Bed:Current，设置以下属性:
    1. Branch: main(要看你仓库的分支)
    2. Custom Url: 加速链接,就是
      https://cdn.jsdelivr.net/gh/ 加上用户名/仓库名(现已失效)
    3. Path: 仓库下照片存储的路径,设置这个目录,随后会在仓库中自动生成该目录
    4. Repo: 用户名/仓库名

踩坑指南

  • 没错，这篇文章就是在VsCode中写的，用起来不错，但是问题也出现了，cdn加速链接已经失效了，这也就导致如果不挂梯，是看不到图片的·····
  • 还可能会报服务端异常，这时检查你的Branch，看看是不是写错了，有的是master而不是main
